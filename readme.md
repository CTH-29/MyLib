在XDU的通院准备电赛时，作为软件组同学，积累了一些芯片驱动，折腾来折腾去感觉还是放在GitHub上好管理一些。

## 使用

所有的上述使用的工具链为

> STM32CubeMX+MDK

大部分驱动使用时需要提前配置STM32的外设，具体查看相应h文件的TIP部分。

为了方便移植，每个驱动尽可能的把与底层相关的内容抽离出来，进行定义或重写即可。
